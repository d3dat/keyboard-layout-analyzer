# В данном файле представлены раскладки, которые
# можно использовать для сравнительного анализа
# при формировании индивидуальной раскладки.
#
# Следует обратить внимание, что в представленных
# раскладках используется разный набор символов,
# поэтому различия в статистике будут обусловлены
# не только расположением символов, но и различиями
# в частотности используемых в раскладках символов.
#
# Также следует иметь в виду, что при использовании
# данного файла для генерации новых раскладок без 
# указания базовой будут использоваться наименее частотные
# буквы из всех представленных раскладок, что может
# привести к выпадению из сгенерированных раскладок
# наиболее частотных букв.
#
# В связи с этим при использования данного файла при
# генерации новых раскладок следует указывать базовую
# раскладку, в этом случае для поиска новых раскладок
# будут использоваться буквы только из указанной
# раскладки.
#
# Или использовать в качестве базового файл layout/english-mod.txt,
# в котором содержатся модифицированными версиями представленных
# раскладок с одинаковым набором символов.
